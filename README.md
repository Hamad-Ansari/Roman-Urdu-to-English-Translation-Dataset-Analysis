This repository contains a comprehensive exploratory data analysis (EDA) notebook for the **Roman Urdu Parallel Translation Dataset**. The dataset consists of 4,000 sentences across multiple categories, with parallel translations in Roman Urdu, Urdu script, and English (both LLM-generated and Google Translate versions).

**Dataset Source:** [Roman Urdu Sentences with Urdu and English Translations](https://www.kaggle.com/datasets/abdulwaliullah/roman-urdu-sentences-urdu-and-english-translations)

## ğŸ“‹ Dataset Structure

| Column | Description |
|--------|-------------|
| `category` | Domain category of the sentence (Health, Technology, Education, etc.) |
| `roman_urdu` | Sentence written in Roman Urdu script |
| `urdu_script` | Sentence written in formal Urdu script |
| `translated_by_llm` | English translation generated by a Large Language Model |
| `translated_by_google` | English translation from Google Translate (for comparison) |
